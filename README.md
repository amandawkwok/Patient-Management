# Patient-Management

A Java-based EE web application that serves to replace manual processing of Patient information and their Appointment information for a doctor's office. Users will be able to perform CRUD operations on specific entities.

## Getting Started

To get the project up and running on your local machine for development and testing...

### Prerequisites

* [Tomcat](http://tomcat.apache.org/tomcat-7.0-doc/)
* [JSTL](http://www.java2s.com/Code/Jar/j/Downloadjstl12jar.htm)
* [MySQL](https://www.mysql.com/downloads/) with tables created from queries found in [tableCreationQueries](https://github.com/amandawkwok/Patient-Management/blob/master/WebContent/resources/tableCreationQuery)
  
  *Credentials in [ConnectionCredentials.java](https://github.com/amandawkwok/Patient-Management/blob/master/src/helper/ConnectionCredentials.java) must be changed to match the user's personal MySQL credentials*
* [Java](https://www.java.com/en/download/)
